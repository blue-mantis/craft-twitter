Setup Instructions

Some section of this example application require specific setup.

Mentions

  • Install Twitter plugin
  • Create a global and call it homepage
  • Create a matrix field and call it Mentions
  • Add a Block Type for this Matrix field, and call it Tweet
  • Add a Tweet field for this Block Type, and call it Tweet as well
  • Now copy some tweet URLs and paste them in the homepage.mentions field so we have some data to play with
